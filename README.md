## CollabNotes
This multi-part, semester-long assignment was concerned with the UI and UX design of a mobile app and people using it. There were parts I did on my own, as a designer, and other parts where I evaluated the work of others. Both aspects mimiced typical design activities from practice. The design that I created is a mobile app for using and keeping academic notes for studying and research. I researched and described typical users of such an app, created interactive scenarios of such users, and created designs of user interfaces. I also received feedback on my first wireframes from my peers and made adjustments accordingly.
